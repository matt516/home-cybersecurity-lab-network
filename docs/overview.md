📌 Project Overview

🛡️ Home Cybersecurity Lab

This project documents the design and implementation of a home cybersecurity lab built to simulate an enterprise network environment.

The lab uses pfSense as a firewall, along with VLAN segmentation and multiple physical and virtual machines, to practice real-world cybersecurity scenarios.

---

🎯 Objectives

* Build a segmented network using VLANs
* Configure firewall rules and access control
* Simulate attacks using Kali Linux
* Analyze network traffic and logs

---

🧱 Environment

* pfSense firewall (dedicated hardware)
* Araknis switch (core switching)
* Raspberry Pi (server)
* Linux machines (targets)
* Virtual machines (Kali, Windows)
